## Simple Array implementation
### Complexity:
 - Space complexity(for n push operations) - **`O(n)`**
 - Time complexity of **_enQueue_** operation - **`O(1)`**
 - Time complexity of **_deQueue_** operation - **`O(1)`**
 - Time complexity of **_Size_** operation - **`O(1)`**
